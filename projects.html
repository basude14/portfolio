  3. Diabetic Retinopathy Detection using Convolutional Neural Network (CNN)

    • Diabetic Retinopathy (DR), an ailment deriving from diabetics is a severe condition that affects the eye. Although there are procedures for detecting DR, they need manual evaluation of ophthalmologist.
    • So, we developed a model which uses Deep Learning for the purpose of this ailment through automation.
    • This model is built with Keras and Tensorflow backend with python and suggested name is “Wide Net X.”
    • Achieved a 92% accuracy which is higher than current system models.
